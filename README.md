# To-dos API
## Starting guide
To create your .env file : 
```sh
cp .env.example .env
```

To install  `node_modules` :
```sh 
npm i
```

To start a database :
```sh 
npm run db
```

To run tests : 
```sh 
npm test
```

To run the app on `localhost` with `nodemon` : 
```sh
npm run dev
```
